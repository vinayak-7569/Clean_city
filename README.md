# CleanCityAI

## AI-Powered Smart Waste Management Platform

CleanCityAI is an intelligent waste management system designed to help municipalities streamline waste complaint handling, improve resource allocation, and optimize waste collection operations using AI-driven insights.

---

## Features

### Citizen Module

* Register and Login
* Submit Waste Complaints
* Upload Waste Images
* Track Complaint Status
* Receive Notifications

### Admin Module

* View All Complaints
* Verify Complaints
* Assign Drivers
* Update Complaint Status
* Generate Hotspots
* Generate Forecasts
* Generate Collection Routes

### Driver Module

* View Assigned Tasks
* Update Task Status
* Complete Waste Collection Jobs

### AI Features

* Waste Category Classification
* Severity Prediction
* Priority Assignment
* Hotspot Detection
* Forecasting
* Route Optimization
